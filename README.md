## Getting Started

### 1. Clone this repository

```bash
git clone https://github.com/tony-hsueh/portfolio.git
```

### 2. Go into the repository

```bash
cd portfolio
```

### 3. Install dependencies

```bash
npm install
```

### 4. Run the app

```bash
npm start
```
